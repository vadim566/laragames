Song 2: The Whole Entire World

Yoorra badlooyoonoo widla mirinya 🔊 The whole world
widla wilgoorndoo bari waloonga garngara. 🔊 is very narrow bridge,
Gooda waiigoomaga! 🔊 but the main thing is not to fear at all.

In Barngarla there is no word for ‘bridge,’ so to say it, we have to work out other ways of describing a bridge. A bridge is pretty much a narrow passage to cross water, in this case literally “widla” (Eng.: “path”) matches the idea. So, when we combine it with “bari” we get “bari widla” which has a similar meaning to the English “bridge.”

Instead of finding a perfect translation, sometimes it’s necessary to describe it as something similar.

Similarly, there is no way to say “the main thing” in Barngarla, so instead the last sentence was completely changed to be a command meaning “Have no fear!” — even though it’s not a word-for-word translation, the idea is still there.

Literally the translation means “the whole earth is similar to a path, a very narrow interval path above a river, but have no fear!”

The “interlinear gloss” looks like this:

Yoorra badlooyoonoo widla mirinya
Earth the-whole path similar
widla wilgoo-rndoo bari waloo-nga garngara.
path narrow-EMPH river interval-LOC above.
Gooda waiigooma-ga!
